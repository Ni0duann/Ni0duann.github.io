#正题：
看了好几种方法，还是hexo结合githubPage最好。

1.下载git和node.js（Windows系统）

这个基本没坑，盲入。安装后一直摁next即可完成。
有一个坑是可能npm命令报错：运行一堆err
找到.npmrc文件删除即可
找不到的先打开隐藏文件可见开关
用npm cache clean –force命令删除好像也行，不过我用了没效果
还有个难受点Win11好像右键隐藏了很多软件的右键功能，导致git右键使用困难。我个人感觉很受影响，例如nodepad++也不方便。解决方法：（参考其他博客的教程 cmd输入
reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
回车，重启

想恢复 cmd输入
reg.exe delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /va /f
回车，重启

2.安装hexo搭建本地博客

（1）安装hexo

安装之前先建立一个空文档用于存放本地博客
右键打开gitbash
输入
npm install -g hexo-cli

（2）初始化hexo，新建存储博客的文件夹

来着hexo官网，按顺序每一步，没坑

（3）使用Netlify部署博客及部署自动化（感觉可以忽略，有需要再搞连续创建了两个账号都立刻被封了说我账号有问题

（4）连接github仓库真的巨多坑，卡了有点久（类目

1.github建立仓库,一定要用github的username去建立（卡了巨久，真的坑，很多博主本来github就创建得好名字，顾直接用一样的，一笔带过了，不仔细看不知道，被这404卡哭了
2.生成SSH添加到GitHub
回到你的git bash中，

在输入，连续摁回车

生成了.ssh的文件夹。在你的电脑中找到这个文件夹
打开完全复制public那个，添加到github中，名称随便；
3.接下来也是坑
有了秘钥就可以用
ssh -T git@github.com
查看是否成功
将hexo部署到GitHub
